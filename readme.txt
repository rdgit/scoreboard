prototype scoreboard app for GGS

design by ISAS team, April 2022

instructions:
a) open display file, using Chrome or Edge browser
b) open dashboard file, set values and press update
b) position display file for scoreboard display
d) update values in dashboard, press update to transfer to display
e) if restart or close of browser, values should stay intact


help:
strike is labelled by clicking radio next to batter
batting history - use enter key to make into list



notes
* firefox browser does not work with localstorage over 2 files
